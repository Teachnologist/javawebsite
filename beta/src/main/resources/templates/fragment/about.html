About

In my day job, I have the opportunity to use all of the PHP, VueJS, Laravel, Red Hat Enterprise Linux, MySQL/MariaDB, Bash, git and other tools and languages to build upon an internationally promoted, enterprise approved learning management system called Elentra. Due to the nature of enterprise security, there isn’t a whole lot I can show on the web. So if you’re interested, please checkout my LinkedIn profile and my GitHub repository to get an idea of what I have been doing for my career.


It is nice to unwrap a prepackaged application and force it into my workflow, but it is even nicer to have the skills to build the components that offer a favorable and worthwhile user experience; so I built this site in IntelliJ with nothing but the the Java Spring Framework and a dream. The html is created via Thymeleaf and Material Design Lite handles the style of the layout. I threw in some jQuery for User Interface control and AJAX processing. One of my favorite things about this site is that javascript is not required for functionality; though it required 2N of my time, I enjoy developing applications that will work on the majority of browsers regardless of the settings.


In the near future, I will be using this site to deploy small applications that have touched some of my pain points. I love interacting with REST APIs, big data, and server installed but client facing productivity tools such as Tableau, and I want to share all of my work with you. Don’t be a stranger; I left my LinkedIn, Github and an email form on this site for us to become great friends in technology.
